OneTabCloud 隐私政策

最后更新时间：2024年5月

1. 引言

欢迎使用 OneTabCloud。我们非常重视您的隐私保护。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

2. 信息收集

我们收集的信息包括：

  • 浏览器标签页数据：用于同步和备份您的标签页信息
  • GitHub Gist Token：用于访问您的私有 Gist 存储库
  • 基本使用数据：用于改进产品功能和用户体验

3. 信息使用

我们使用收集的信息：

  • 提供标签页同步和备份服务
  • 通过 GitHub Gist API 访问您的私有存储库
  • 改进产品功能和用户体验
  • 解决技术问题

4. 数据存储

您的数据存储在您自己的本地和 GitHub Gist 私有存储库中。您需要提供 GitHub Gist Token 来授权访问。我们不会在任何其他服务器上存储您的数据。所有的数据同步和备份操作都是通过 GitHub Gist API 直接与您的私有存储库进行交互。

5. 数据共享

我们不会出售、交易或转让您的个人信息给第三方。您的数据完全存储在您自己的本地和 GitHub Gist 私有存储库中，您可以完全控制数据的访问权限。

6. 用户权利

您有权：

  • 访问您的个人数据
  • 更正不准确的数据
  • 删除您的数据
  • 导出您的数据

7. 数据安全

我们采取适当的技术和组织措施来保护您的数据安全，包括：

  • 数据加密
  • 访问控制
  • 定期安全审计

8. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并更新"最后更新时间"。
